# FPGA-Implementaion-of-Spiking-Neural-Network-For-Patern-Recognition
This repository contains the source code, hardware design, and documentation for my M.Tech final-year project. The project focuses on implementing a hardware-efficient Spiking Neural Network (SNN)-based Convolutional Neural Network (CNN) accelerator on FPGA to achieve low-power and high-performance neural network inference.

Project Objectives
Design an FPGA-based SNN-CNN architecture.
Implement neuron models using Verilog HDL.
Develop convolution, activation, and pooling modules.
Optimize the design for low power consumption and reduced hardware complexity.
Verify functionality through simulation and FPGA implementation.
Features
Verilog HDL implementation
FPGA-compatible architecture
Hardware-efficient neuron models
Convolution and Max Pooling modules
Modular and scalable design
Simulation and synthesis support
